# 📊 Olist E-Commerce Data Analysis

## 📌 Overview

This project performs **Exploratory Data Analysis (EDA)** on the **Olist Brazilian E-Commerce Dataset** using Python. The goal is to uncover valuable business insights by analyzing customer behavior, sales trends, product performance, payment methods, delivery efficiency, and customer reviews.

---

## 🎯 Objectives

- Clean and preprocess raw data
- Analyze customer purchasing behavior
- Identify sales trends
- Study product category performance
- Analyze payment methods
- Evaluate delivery performance
- Generate business insights through visualization

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Jupyter Notebook

---

## 📂 Dataset

The project uses the **Olist Brazilian E-Commerce Public Dataset**.

Datasets included:
- Customers
- Orders
- Order Items
- Products
- Sellers
- Payments
- Reviews

---

## 📈 Analysis Performed

- Sales Trend Analysis
- Customer Analysis
- Product Category Analysis
- Payment Method Analysis
- Delivery Time Analysis
- Customer Review Analysis

---

## 📊 Key Insights

- Identified monthly sales trends.
- Found top-selling product categories.
- Analyzed customer purchasing patterns.
- Compared different payment methods.
- Measured delivery efficiency.
- Evaluated customer satisfaction through review scores.

## 🚀 Installation

Clone the repository:

```bash

```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the notebook:

```bash
jupyter notebook Olist_Ecommerce_Analysis.ipynb
```
